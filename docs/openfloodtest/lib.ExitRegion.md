## Function: `Lib.Script.ExitRegion`
- Unlocks the exit
> Server Only, If you are using the Pfe Lib you also use ``Lib.ExitRegion``
### Parameters:
- `ForceExit` (bool): exit open?
- `ButtonActive` (bool): is the buttons active?
### Returns:
- `The Function itself`.
## Example
```lua
Lib.Script.ExitRegion(true,true)
```
